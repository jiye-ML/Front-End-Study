## JavaScript 类型运算符

| 运算符     | 描述                                  |
| :--------- | :------------------------------------ |
| typeof     | 返回变量的类型。                      |
| instanceof | 返回 true，如果对象是对象类型的实例。 |

**注释：**[JS 类型转换](https://www.w3school.com.cn/js/js_type_conversion.asp)这一章完整描述了类型运算符。
