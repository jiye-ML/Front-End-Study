除了英文单词 red，我们还可以使用十六进制的颜色值 #ff0000：

```
p { color: #ff0000; }
```

为了节约字节，我们可以使用 CSS 的缩写形式：

```
p { color: #f00; }
```

我们还可以通过两种方法使用 RGB 值：

```css
p { color: rgb(255,0,0); }
p { color: rgb(100%,0%,0%); }
```

请注意，当使用 RGB 百分比时，即使当值为 0 时也要写百分比符号。但是在其他的情况下就不需要这么做了。比如说，当尺寸为 0 像素时，0 之后不需要使用 px 单位，因为 0 就是 0，无论单位是什么。
